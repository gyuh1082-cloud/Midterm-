Inclusive Harlem Plaza

Redesigning Harlem Pedestrian Plaza for comfort, dignity, and shared public use.

About the Project

This project redesigns Harlem Plaza to support a wide range of users by improving comfort, accessibility, dignity, and shared use.

Essential Question

How can Harlem Plaza be redesigned so it feels more welcoming and usable for diverse users?

The Problem

  • Lack of comfortable seating
  • No shade or weather protection
  • Poor accessibility
  • Weak safety and visibility

Problem Statement: The plaza does not support inclusive, dignified use for diverse users.

Site / Context

User Groups

  • Elderly residents
  • People with disabilities
  • Families
  • Commuters

Key issues include unclear circulation, uneven comfort, and user conflicts.

Design Options

Comfort First Plaza

Focus on seating, shade, and quiet zones.

Shared Use Plaza

Flexible space for events and daily use.

Community Support Plaza

Spaces designed for dignity and support.

Final Proposal

The final design combines comfort, accessibility, and shared use.

  • Tiered seating
  • Wheelchair accessibility
  • Shade structures
  • Flexible social spaces
